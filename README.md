## Our Stack

- [NextJS](https://nextjs.org/)
- [React](https://reactjs.org/)
- [Framer Motion](https://www.framer.com/motion/)
- [Tailwind](https://tailwindcss.com/)
- [Wagmi](https://wagmi.sh/)
- [ConnectKit](https://docs.family.co/connectkit)

## Getting Started

```bash
yarn
yarn dev
```
